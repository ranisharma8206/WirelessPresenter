Running server side application
-> Convert ppt to images and number the images as 1.jpg 2.jpg ....  and place them in slides folder
-> Edit  Notes.json in slides folder to add custom notes, these will be synced with the andriod appliaction

Running python application
$ python3 ./main.py

Compiling client side application
Install android studio
install NPM and using npm install react-native and expo-cli

Run the following commands.
$ expo install
$ expo android

 
 
Compiling gyroscope version
Install Andriod studio
Install phonegap
create account on phonegap build for compiling.

$ phonegap install

after calling above command, create zip of application and upload on phonegap build server, the server outputs .apk file install it on client.
